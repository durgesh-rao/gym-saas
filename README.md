# Gym SaaS

A Gym Management System built with Laravel and Vanilla JavaScript.

## 🚀 Features

- User Authentication with Laravel Sanctum
- Gym Management (CRUD)
- Plan Management (CRUD)
- Member Management (CRUD)
- RESTful APIs
- Role-Based Access Control
- Pagination
- Search Functionality


## 🛠️ Tech Stack

- Laravel
- PHP
- MySQL
- Laravel Sanctum
- HTML
- CSS
- Bootstrap
- Vanilla JavaScript
- XAMPP

## 📂 Project Structure

- Authentication
- Gym Module
- Plan Module
- Member Module
- API Integration
- Frontend (HTML + JavaScript)

## ⚙️ Installation

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev
php artisan serve
```
 Start Apache and MySQL using XAMPP. <br>
 http://localhost/gym-saas/public/frontend/login.html
```
